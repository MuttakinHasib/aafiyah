### Prerequisites

This is an example of how to list things you need to use the software and how to install them.


### Installation

1. Clone the repo
   ```sh
   git clone --recurse-submodules https://github.com/muttakinhasib/aafiyah.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   or
    ```sh
    yarn
   ```
